In order to add a new local machine for executing Experiments, please follow the below steps:
  1. Download script with the following command: 
    wget https://raw.githubusercontent.com/deepcortex/dc-local-installer/develop/deploy.sh -O deploy.sh
  2. Run deploy.sh script using bash: 
    $ bash deploy.sh
  3. You will be required to enter your DeepCortex username and password as well as a "machine name" to identify the given machine within your DeepCortex account.
